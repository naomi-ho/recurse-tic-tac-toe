# Tic Tac Toe

A tic tac toe game that you can play online with a friend. Each player takes turns clicking on each cell until someone wins the game or the game ends in a tie. Then the text will flash indicating that the game has ended with the results. There's a Restart button that can be clicked at any time to start a new game.

[Live Demo](https://naomi-ho.github.io/recurse-tic-tac-toe/)
